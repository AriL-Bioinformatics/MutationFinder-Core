# Genetic Mutation Finder

This is a Java based program to find mutation in DNA sequences.

## Structure:
- CompareDNA
- Find mutation location
- Object-Oriented implementation 

## In Progress 
- Detect Mutation Type 
	- Insertions
	- Deletions
 	- Substitutions 

## Walkthrough
1. Run main java file 
2. Input two sequences 
3. Program compares sequences and identifies any mutations 
4. Results are displayed in the console 

# Motivation 
This project was developed to apply data structures and object-oriented programming concepts while exploring bioinformatics and genetic sequence analysis.
